Automation? Yes! Whoop! Whoop!

With the printer you can automate the creation of the bit blocks.

It will draw bits from the bit storage blocks next to it, but only from the side which are not its front (you need to be able to control it still).

How to use it, you ask?

Well simple: Make a sealed pattern, put it with a chisel inside your printer. And fill up the storages with the requested bits of choice!
